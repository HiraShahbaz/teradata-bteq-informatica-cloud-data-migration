# Data Migration Between Servers Using BTEQ Scripts in Informatica Cloud

## Overview
This repository documents an enterprise data migration solution implemented using "BTEQ (Basic Teradata Query)" scripts orchestrated via "Informatica Intelligent Cloud Services (IICS) Command Tasks".The solution demonstrates how pre-processing logic can be used to safeguard target Teradata tables before loading new data from source systems.

All sensitive information such as credentials, hostnames, and client identifiers has been anonymized.


## Technology Stack
- Informatica Intelligent Cloud Services (IICS)
- Secure Agent
- Teradata Database
- BTEQ (Basic Teradata Query)


## Repository Structure

- **/document**  
  Final implementation document in PDF format, including step-by-step configuration and execution details.

- **/screenshots**  
  Screenshots from the original implementation showing task creation,configuration, execution, and validation.

- **/scripts**  
  Sanitised sample BTEQ script used as a pre-processing command.
